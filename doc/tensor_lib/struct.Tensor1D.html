Struct tensor_lib::Tensor1D

source · 
pub struct Tensor1D<T> {
    pub data: Vec<T>,
    pub a: usize,
}
Expand description

1d tensor with [dynamic] dimensions.

self.data contains the underlying data.

If present the properties a, b, c, etc. represent the lengths of dimensions 0, 1, 2, etc.

impl<T> Tensor1D<T>

source

pub fn from_distribution<DIST: Distribution<T>>( (a, distribution): (usize, DIST) ) -> Self

Constructs a tensor of a given shape with values sampled from a given distribution.

use rand::distributions::{Uniform,Standard};
use tensor_lib::Tensor1D;
let x = Tensor1D::<i32>::from_distribution((2,Uniform::<i32>::from(0..10)));
let y = Tensor1D::<f32>::from_distribution((2,Standard));
